# 📰  Dragon News

A modern and responsive news portal web application built with React.  
This project was my first experience implementing authentication and protected routes, helping me understand real-world frontend application structure and user management.

## ✨ Features

- 🔐 Firebase Authentication
  - Email & Password Login/Register
  - Google Sign-In
  - Protected/Private Routes

- 📰 Dynamic News System
  - Category-based news filtering
  - News details page
  - Responsive news cards

- 🎨 Modern UI/UX
  - Fully responsive design
  - Clean layout using Tailwind CSS
  - Interactive components

- ⚡ Routing & Navigation
  - React Router DOM
  - Dynamic routes
  - Error handling routes

- 📱 Mobile Friendly
  - Optimized for desktop, tablet, and mobile devices

🔑 Environment Variables

Create a .env.local file in the root directory and add:

VITE_apiKey=your_api_key
VITE_authDomain=your_auth_domain
VITE_projectId=your_project_id
VITE_storageBucket=your_storage_bucket
VITE_messagingSenderId=your_sender_id
VITE_appId=your_app_id
⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/your-username/dragon-news-client.git

Go to the project folder:

cd dragon-news-client

Install dependencies:

npm install

Run the development server:

npm run dev
